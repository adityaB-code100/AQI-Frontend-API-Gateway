## Overview
This is a comprehensive Air Quality Index (AQI) monitoring system that serves as the **Frontend and API Gateway** for the AQI project. The application provides real-time AQI data, health alerts, and personalized recommendations based on user profiles and location.

> **Note:** The machine learning backend for AQI predictions is available in a separate repository: [AQI-ML-Backend](https://github.com/adityaB-code100/AQI-ML-Backend/)
